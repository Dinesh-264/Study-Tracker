# 🎯 Study Tracker Pro

A lightweight, mobile-first **Progressive Web App (PWA)** designed to track daily study hours with a clean, goal-oriented interface.

## ✨ Features

*   **Daily Progress Tracking:** Easily log 0.5hr and 1.0hr study sessions.
*   **Dynamic Goals:** Visual feedback changes to **Gold** once the 3-hour daily target is reached.
*   **Motivational "Toasts":** Pop-up celebrations and quotes appear when you push past your limits.
*   **Weekly History:** Automatically archives your sessions every Saturday to track long-term consistency.
*   **Monthly Stats:** View how many days you've hit your target relative to the current month.
*   **Privacy Focused:** All data is stored locally on your device using `localStorage`. No accounts or servers required.

## 📱 How to install on your phone

This website is designed to be used as a "Home Screen App."

### Android (Chrome)
1. Open the website URL in Chrome.
2. Tap the **three dots (⋮)** in the top right.
3. Select **"Add to Home Screen"**.
4. The tracker will now appear in your app drawer and hide the browser bars when opened!

### iOS (Safari)
1. Open the website URL in Safari.
2. Tap the **Share** button (square with arrow).
3. Scroll down and tap **"Add to Home Screen"**.

## 🛠️ Built With

*   **HTML5** - Structure & PWA Meta tags.
*   **CSS3** - Custom animations, glassmorphism, and mobile-responsive layout.
*   **Vanilla JavaScript** - Logic, state management, and local storage.

## ⚠️ Important Note
Since data is stored in the browser's `localStorage`, clearing your browser cache or deleting the site data will reset your history. Use the **"Clear All Data"** button in the History tab if you want a fresh start.
